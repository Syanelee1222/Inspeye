# Inspeye 灵感桌宠
A tool helpful for artists and designers to gain inspiration. In a desktop pet form that stays on your screen, it can aggregate RSS content sources.<br>
Simply hover your mouse over the desktop pet to bring up a sleek panel for browsing the latest cases.<br>
一款专为艺术设计工作者获取灵感的支持工具，以桌宠形态常驻桌面，可聚合RSS内容源。<br>
只需将鼠标移至桌宠上方，即可弹出精美面板浏览最新设计案例。
# WIP 仍在开发中

该项目仍在持续测试并优化用户体验中。
